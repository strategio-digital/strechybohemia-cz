# Strechybohemia.cz
Frontend for [strechybohemia.cz](https://www.strechybohemia.cz) website based on Contentio-SDK

## Installation guide
1. `cp .env.example .env`
2. `php -S localhost:8080 -t www/`
3. `composer i`
4. `composer analyse`
5. `yarn && yarn dev`
6. [http://localhost:8080](http://localhost:8080)
