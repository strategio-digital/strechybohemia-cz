# Strechybohemia.cz
Frontend for [strechybohemia.cz](https://www.strechybohemia.cz) website based on Contentio-SDK

## Installation guide
1. `cp .env.example .env`
2. `./project.sh serve --stop-others`
3. `./project.sh app`
4. `composer i`
5. `composer analyse`
6. `yarn && yarn dev`
7. [http://localhost:8080](http://localhost:8080)
