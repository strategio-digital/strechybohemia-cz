# Strategio.digital
Frontend for [strategio.digital](https://strategio.digital) website based on Contentio-SDK

## Installation guide
1. `cp .env.example .env`
2. `./project.sh serve --stop-others`
3. `./project.sh app`
5. `composer i`
6. `composer phpstan`
4. `yarn && yarn dev`
7. [http://localhost:8080](http://localhost:8080)
